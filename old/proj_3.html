Anime Recommender App

This is my personal desktop app I created to get recommendations for what to watch. I used Java Swing for the GUI and two APIs to get anime data. I tried to implement OOP techniques such as encapsulation and abstraction. The main challenge I faced when developing this app was working with JSON data and figuring out the layout.

Images:

The user is able to enter their MyAnimeList username so that their anime history can be retrieved.



After logging in the user can select how they would like the recommendations to be generated.
